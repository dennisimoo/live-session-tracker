# Live Session Tracker

Import and see live user sessions in real-time.

## Quick Start

```bash
npm start
```

Then add to your site:

```html
<script src="http://localhost:3001/src/tracker.js"></script>
```

That's it. View dashboard at: **http://localhost:3001/dashboard.html**
